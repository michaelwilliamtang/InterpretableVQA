# InterpretableVQA
Code for the paper [Evaluating Interpretability Techniques with Human Attention in Visual Question Answering](https://michaelwilliamtang.github.io/blog/papers/InterpretableVQA.pdf).

This is a joint work with Jason Yuan and served as the final project for [COS429](https://registrar.princeton.edu/course-offerings/course-details?term=1222&courseid=009123) at Princeton.
